REACT QUERY HAS BUILT-IN  STATES;     https://react-query.tanstack.com/reference/useQuery

Destructuring useQuery() can get automatic values from fetching data using fetch or axios 
most importants are:
- data   //data to create a map
- status
- error  //type of error
- isError  //boolean true/false
- isLoading  //boolean true/false
- isLoadingError
- isSuccess

example:    
const {data, isLoading, error, isError} = useQuery("fecthPics", functionFetchPics);
