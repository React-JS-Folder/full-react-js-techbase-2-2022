REACT JS - EXTENDING STYLE FROM AN ALREADY CREATED STYLED-COMPONENT ELEMENT 

1. Create the element you want to extend 
import styled from 'styled-component'

2. Create the component (element) you want to extend 

   const StyledBasicButton = styled.button`
      background-color: ${ (props) => props.primary ? "red" : "green")};
   `;

3. Create a component (element) from it 

   const StyledLargeButton = styled(StyledBasicButton)`
      padding: 1rem;
   `
