if you want to destructure the Context.Provider value 

create an object  -- inside value={ }  ---> add an extra { }  and after you can destructure the useContext
example: 

import {ContextPro} from './Contexts/ContextPro'


to get 1 value from Provider 
Provider) value={username}  ---  Consumer   const username = useContext(ProviderName) 


to get multiple values from Provider 
Provider)  value{{}}     value={{val1, val2, val3}}   --- Consumer const {val1,val2,val3} = useContext(ProvName)


return (
   <ContextPro.Provider value={{variable1,variable2,function1,array}}  
)
