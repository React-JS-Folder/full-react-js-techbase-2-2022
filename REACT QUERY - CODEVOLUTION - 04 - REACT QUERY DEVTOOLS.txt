REACT QUERY - DEVTOOLS 
React Query Devtools is a tool just like google developer tools

1. import { ReactQueryDevtools } from 'react-query/devtools'
2. add it before the closing </QueryClientProvider>

example:

<QueryClientProvider client={anything} >


   <ReactQueryDevtools initialIsOpen={false} position='botton-right' />
</QueryClientProvider >
